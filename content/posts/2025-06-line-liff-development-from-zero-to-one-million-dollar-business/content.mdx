## 什麼是 LINE LIFF？認識你的創業利器

LINE LIFF 是 LINE 提供的前端框架平台，讓開發者能夠在 LINE 聊天室內直接運行網頁應用程式。簡單來說，LIFF 就像是 LINE 的內建瀏覽器，但它的威力遠不止於此。

### LIFF 的核心優勢

1. **無縫體驗**：用戶無需離開 LINE 即可完成各種操作
2. **用戶資料取得**：可獲取用戶的 LINE ID、個人資料等（需用戶授權）
3. **訊息發送**：可代替用戶發送訊息到聊天室
4. **原生功能支援**：支援相機、地理定位、QR Code 掃描等手機功能
5. **跨平台相容**：支援 iOS、Android 及桌面版瀏覽器

## 技術實作指南

### 開發環境建置

```javascript
// 1. 安裝 LIFF SDK
npm install @line/liff@latest

// 2. 初始化 LIFF
import liff from '@line/liff';

liff.init({
  liffId: 'YOUR_LIFF_ID'
}).then(() => {
  // LIFF 初始化成功
  if (liff.isLoggedIn()) {
    // 取得用戶資料
    liff.getProfile().then(profile => {
      console.log(profile.displayName);
      console.log(profile.userId);
    });
  }
});
```

### 核心功能實作

1. **用戶認證**
   - LINE Login 整合
   - 權限請求管理
   - Token 處理

2. **訊息發送**
   - 發送文字訊息
   - 分享至聊天室
   - Rich Message 設計

3. **原生功能**
   - 相機與相簿存取
   - GPS 定位
   - QR Code 掃描
